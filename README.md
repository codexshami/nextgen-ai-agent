##  Overview

**NextGen AI Agent** is a scalable and modular AI-powered system designed to integrate intelligent automation into modern applications.  
It uses **FastAPI**, clean backend architecture, and Docker for easy deployment.

---

##  Features

-  AI Agent Architecture
-  FastAPI Backend
-  Dockerized Setup
-  Scalable & Modular Codebase
-  Pydantic Schema Validation
-  Easy API Integration

##  Getting Started

###  Clone the Repository

```bash
git clone https://github.com/codexshami/nextgen-ai-agent.git
cd nextgen-ai-agent
```

---

###  Run with Docker

```bash
docker-compose up --build
```

---

###  Access API

```
http://localhost:8000
```
